- [ ] **Step 1: Read the code and spec**

Read `docs/SPEC.md` first, then skim `index.html`, then `app.js`, then `style.css`. Note element IDs / classes for each control so proposals can reference them.

- [ ] **Step 2: Triage findings**

For every finding anchored as `D<n>`, `M<n>`, or `I<n>` across the previous sections, decide one of:
- **Propose a fix** — the finding is concrete enough that you can describe the fix in one paragraph.
- **Observation only** — the finding is real but the right fix is a judgement call for the user to make.
- **Defer** — the finding implies a larger redesign (e.g. "the whole toolbar is incoherent"). If you propose a toolbar-level fix, scope it explicitly and call out which findings it subsumes.

- [ ] **Step 3: Write proposals**

For each "Propose a fix" finding, write a P-entry:

```markdown
### P1. Reset button — icon + scope split

**Addresses:** D1, I1, M-cross-ref.

**Proposal:**
- Icon: Lucide `corner-up-left` (clearly "go back / undo") instead of the current ⟲.
- Tooltip: "Reset view (latest run)".
- Behaviour split (optional, lower priority): split the multi-purpose reset into two — one resets the view, the other clears the active match set. Today they're fused; this is the main reason the icon can't be specific.

**Why this addresses the finding:** The current glyph reads as "refresh data", which is the wrong mental model. A directional back-arrow icon plus a more specific tooltip narrows the meaning. The optional split addresses the deeper overload but is a bigger change.
```

If the proposal involves a layout change, include either an ASCII sketch in the markdown or a small mockup image saved as `docs/img/ux-audit/mockups/<descriptor>.png` and referenced inline. Mockups can be quick — taking a screenshot of the live UI and annotating it is fine; building HTML mockups is overkill unless the change is substantial.

If proposing a unified icon set, name the set explicitly (Lucide, Phosphor, Heroicons) and list the specific glyphs per control in a table:

```markdown
| Control | Current | Proposed (Lucide) |
|---|---|---|
| Zoom + | Leaflet text "+" | `plus` |
| Reset | ⟲ | `corner-up-left` |
| Polygon | leaflet-draw default | `pentagon` |

| Settings | ⚙ | `settings` |
| Filter | funnel | `sliders-horizontal` |
```
